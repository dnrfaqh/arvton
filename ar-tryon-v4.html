✅ Rekomendasi Ukuran
🧬 Pipeline Computer Vision
01 BlazePose Pose Estimation
Deteksi 33 keypoint tubuh real-time. Basis dari seluruh pipeline.
MediaPipe BlazePose · modelComplexity=1
02 Body Segmentation
Memisahkan siluet tubuh dari background untuk occlusion handling (baju tidak menimpa area non-tubuh).
MediaPipe Selfie Segmentation · smoothSegmentation=true
03 Distance Estimation
Estimasi jarak user ke kamera berdasar model pinhole camera. D = (Hreal × f) / Hpixel
Pinhole Camera Model · FOV ≈ 60°
04 Anthropometric Measurement
Ekstraksi 4 ukuran tubuh dari keypoint dengan Moving Average Filter (n=10) untuk stabilitas.
Euclidean Distance · Temporal Smoothing
05 Homography 4-Point
Perspective warping 4-titik agar desain mengikuti perspektif tubuh. Solusi DLT melalui Gaussian elimination.
Direct Linear Transform (DLT) · 8×8 System
06 Non-Rigid Mesh Warping
Deformasi non-rigid menggunakan grid mesh 8×8. Kontrol point tambahan (siku, pinggul) dengan bobot Gaussian menghasilkan TPS-inspired warp.
RBF Gaussian Weighting · Triangle Rendering
07 HSV Color Calibration
Sampling brightness rata-rata scene, lalu menyesuaikan desain agar cocok dengan kondisi pencahayaan lingkungan.
Scene Brightness Sampling · CSS Filter
08 Occlusion Clipping
Overlay desain di-clip ke area tubuh hasil segmentasi. Mencegah "bocor" ke background.
destination-in Composite · Mask-based
